
# Refactoring Sistem Validasi Registrasi Mahasiswa

Repository ini berisi contoh kode sebelum dan sesudah refactoring
menggunakan prinsip SOLID (SRP, OCP, dan DIP).

## Struktur Folder
- before_refactoring  : Kode awal (melanggar SOLID, menggunakan if/else)
- after_refactoring   : Kode hasil refactoring (SOLID)

## Tujuan
Menunjukkan penerapan SRP, OCP, dan DIP dalam refactoring kode OOP Python.
